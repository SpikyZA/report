# Report

Report is a Discord bot that host a report management system for your server.

  - Reports
  - Moderation
  - Suggestions

## Prerequisites

- A MongoDB Database (you can get a free tier [here](https://www.mongodb.com/cloud/atlas))
- Discord.js ^v12
- Discord Bot Token
